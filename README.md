# CRUD-sencillo-NodeJS
Repositorio central del proyecto de Desarrollo Web Tecleras Virtuales.

##Instalación
instalar nodejs

en la consola:

cd \<carpeta\>

npm install

npm install -g gulp

##Arrancar
gulp

##Base de datos
Instalar MySQL desde la página respectiva.
La base de datos es remota, así que no tienen que instalarla :D
